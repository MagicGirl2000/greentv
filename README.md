## 简介

跨境双端系统：**深圳**（采样 + 显卡解码分析全球电视频道的视频+音频维度）⇄ **英国·伦敦**（聚合展示、按需视频点播、对外网站）。

- 全球电视频道（iptv-org 174 国 + 中国卫视/地方台）实时“维度”映射，聚合成综合 **绿太阳指数 GreenIndex**，Gate 风格红绿蜡烛展示（绿涨红跌·无变化=白烛）。
- **世界地图（国家级 / 全球省级行政区）**：3139 个省逐省四色降雨判定；频道→城市/省定位（GeoNames），品牌频道归首都（总部）。
- **降雨验证**：以官方天气 API 为地面真值，对比 GreenIndex 指数与 NASA 卫星云图、Open-Meteo 官方降水，生成一致性 Token 与正确率。
- 6 语言网站、按需点播播放器（flv.js）、卫星云图、天气预测。
