# 绿太阳指数 GreenIndex · 央视频道维度检测

> ⚠️ **本指数不可交易**。私下涉赌为违法犯罪。本站为**虚构演绎·仅供音视频分发的简易检测**，结果不科学、可能错误，**切勿当真**。

对央视各频道的**音频**做实时“维度”映射（复用界域分析仪的声学特征库 + 界域字典），聚合成一个像恒生指数那样的**综合绿太阳指数（GreenIndex）**，以 Gate 风格的红绿蜡烛展示（**绿涨红跌·无变化=白烛**）。提供分频道维度、初始维度记录、各时间段（10s/30s/1分/5分/15分/1时）K线、实时建议。

## 意义
对音视频分发做一个**简易、象征性的检测**——虽不可当真，但能直观看出“出现的任何异常/突变”，供人工进一步甄别。

## 运行
```
D:\ballbs\.venv\Scripts\python.exe D:\ballbs\greentv\server.py
# 浏览器打开 http://127.0.0.1:8780/
```
依赖：flask、numpy、imageio_ffmpeg（已在 .venv）。

## 真实检测 vs 演示
- 默认 **演示模式**：各频道走合成的有界随机走势（标“演示”），**不代表任何真实内容**，仅用于展示界面。
- 如需**真实检测**：在 `channels.py` 给对应频道的 `stream` 填入**你自己有权访问的** HLS/流地址；服务端会用 ffmpeg 抓取约3秒音频 → 特征 → 最近邻匹配歌典 → 维度。
- 本项目**不抓取、不内置任何无权访问的受保护流**。

## 结构
```
greentv/
  server.py      Flask + 后台采样worker + K线分桶API（/api/channels /api/series）
  dimension.py   音频→维度（复用 realm_audio 特征库 + realm_data 界名）+ 演示走势 + 建议
  channels.py    央视频道清单（stream 默认留空=演示）
  static/        index.html / app.js（KLineChart 红绿白蜡烛）/ style.css
```
